# Praticando Spring Boot: Challenge LiterAlura

## Objetivo:

* É semelhante a uma biblioteca virtual, que o usuário poderá fazer interações com a aplicação.

## Funcionalidades: 

* Pode-se buscar livro por título;
* Listar todos os livros registrados no banco de dados;
* Listar todos os autores já buscados;
* Mostrar autores vivos em um determinado ano;
* Listar livros em um idioma específico;
